Conectamente
primeiro comint
===



Step by Step:



* Primeiro devemos baixar a ferramenta **Git** através do link https://git-scm.com/install/windows;
* Após baixar e instalar o Git, devemos clonar o repositório **remoto do GitHub** para o **repositório local** (arquivo 'clone repositorio' indica onde pegar o link para baixar);
* O comando para clonar é 'git clone {link gerador}' (arquivo git clone);
* Após ter o repositório **local** fazer um teste **commitando** alguma coisa no arquivo do README;
* Após adicionar algum texto, deve-se enviar para o repositório remoto no GitHub:


git add .				        (adicionar as alterações)
git commit -m "meu commit"		(adicionar o comentário das alterações)
git push				        (enviando para o gitHub)
