Conectamente
primeiro comint
===



Step by Step:



* Primeiro devemos baixar a ferramenta **Git** através do link https://git-scm.com/install/windows;
* Após baixar e instalar o Git, devemos clonar o repositório **remoto do GitHub** para o **repositório local** (arquivo 'print 1) indica onde pegar o link para baixar;
* O comando para clonar é: 'git clone {link gerador}' (print 2);
* Após ter o repositório **local** fazer um teste **commitando** alguma coisa no arquivo do README (print 3);


git add .				        (adicionar as alterações)
git commit -m "meu commit"		(adicionar o comentário das alterações)

* Após adicionar algum texto, deve-se enviar para o repositório remoto no GitHub: (print 3)
git push				        (enviando para o gitHub)


- quando for fazer pela primeira vez o commit, provavel que tenha que configurar o email e username, só seguir o exemplo que o proprio terminal do git informa. (print 4) só se faz uma vez.
